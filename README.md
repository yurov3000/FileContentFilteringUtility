Утилита фильтрации содержимого файлов.

Инструкция:
1) Скачиваете программу, распаковываете
2) Заходите в терминале в папку куда распаковали
3) вводите ./gradlew build (ждёте установки)
4) вводите java -jar build/libs/FileContentFilteringUtility-1.0-SNAPSHOT.jar [флаги] [префикс] "[путь из какого файла читаем]" "[путь куда сохраняем]"


Пример команды запуска:

java -jar build/libs/FileContentFilteringUtility-1.0-SNAPSHOT.jar -s -a -p sample- in1.txt in2.txt



Описание флагов:

-o : Задаем путь сохранения

-p : Префикс к файлу сохранения

-a : Режим добавления в текущие файлы (дефолт: файлы перезаписываются)

-s : Краткая статистика

-f : Полная статистика
